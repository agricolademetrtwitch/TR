# 🤖 Steam Autonomous Godmode Suite v2026

Полностью автономный веб-сайт и многофункциональный комбайн на **Node.js и SQLite**, разработанный для круглосуточного (24/7) мониторинга, автоматического фарма коллекционных карточек, буста игровых часов, управления достижениями и интеллектуальной защиты инвентаря.

Проект спроектирован по монолитному принципу отказоустойчивости (Fail-Safe), оптимизирован под облачные хостинги (Render, Railway) и не требует создания сторонних папок или раздельных HTML-файлов.

---

## 🛠 Ключевой Функционал Системы

*   **👥 Пул Аккаунтов (Архитектура ASF):** Добавление неограниченного количества Steam-ботов через веб-интерфейс. Каждая сессия работает в изолированном потоке памяти с защитой от утечек ОЗУ.
*   **🎮 Автофарм Карточек и Накрутка Часов:** Полноценный параллельный запуск до 5 популярных AppID одновременно (CS2, TF2, Dota 2 и др.) для генерации игрового времени и дропа карточек 24/7.
*   **🎯 Менеджер Достижений (SAM Picker):** Встроенный модуль инжекции Protobuf-пакетов для мгновенной и безопасной разблокировки 100% достижений в любой игре прямо из панели сайта.
*   **🔑 Лицензионный DLC Unlocker:** Модуль автоматической записи поддельных лицензионных подписей во внутренний реестр сессии для открытия платного скрытого контента.
*   **🛡 Сетевой Анти-API Скам (Advanced Trade Manager):** Постоянный мониторинг входящих обменов. При обнаружении попытки скрытого или одностороннего вывода вещей (API-Scam мошенничество) трейд моментально блокируется.
*   **🤖 Интерактивный ИИ-Терминал (Walftech Engine):** Мультиязычная консоль для управления установками, генерации чистых файлов конфигурации Steam AppManifest (`.acf`) и скриптов на языке Lua.
*   **🔒 Мастер-Пароль и Уведомления:** Доступ к панели защищен сквозным мастер-паролем. Интегрирован шлюз для отправки экстренных push-уведомлений на ваш телефон через Telegram-бота при фиксации атак на инвентарь.

## 📂 Архитектура Репозитория

Проект имеет монолитную структуру, что исключает ошибки путей `ENOENT` (файл не найден) при деплое в облачные контейнеры:

```text
├── server.js               # Монолитное ядро бэкенда со встроенным HTML5/CSS3/JS фронтендом
├── package.json            # Спецификация зависимостей, оверрайдов и скриптов запуска
└── README.MD               # Инструкция и документация репозитория (этот файл)
```

## 🌐 Пошаговая инструкция по развертыванию в Облако (Render.com)

Проект полностью оптимизирован под сетевые стандарты и изоляцию облачных сред (динамические порты `process.env.PORT` и перенос БД в директорию `/tmp` для обхода read-only ограничений хостинга).

1. Загрузите файлы `server.js` и `package.json` **прямо в корень** вашего репозитория на **GitHub**.
2. Зарегистрируйтесь на сайте **Render.com** с помощью вашего GitHub-аккаунта.
3. В панели Render нажмите кнопку **New +** в верхнем углу страницы и выберите **Web Service**.
4. Выберите из списка ваш репозиторий с ботом и нажмите кнопку **Connect**.
5. Укажите следующие параметры конфигурации в полях настройки:
    *   **Name:** `любое-уникальное-имя-вашего-сайта`
    *   **Region:** `Frankfurt (EU)` (Рекомендуется для минимального пинга к серверам авторизации Steam)
    *   **Language:** `Node`
    *   **Root Directory:** Оставьте **полностью пустым** (blank)
    *   **Build Command:** `npm install`
    *   **Start Command:** `node server.js`
    *   **Instance Type:** `Free` (Бесплатный тариф)
6. Прокрутите страницу вниз и разверните блок **Advanced**. Нажмите кнопку **Add Environment Variable** и добавьте пароль для защиты вашего сайта:
    *   **Key:** `ADMIN_PASSWORD`
    *   **Value:** *Придумайте ваш секретный пароль от сайта* (Если поле не заполнять, по умолчанию в коде применится пароль `ADMIN1234`).
7. Нажмите синюю кнопку **Deploy Web Service** в самом низу страницы.

Через 1–2 минуты сборка завершится, статус проекта перейдет в зеленый маркер **Live**, а хостинг выдаст вам защищенную публичную `https://` ссылку на вашу персональную панель управления.

## 💻 Использование Интерактивного Терминала

Для отправки команд перейдите на вкладку **«🤖 ИИ-Терминал»**, введите команду в строку ввода внизу панели и нажмите **Enter**:

*   `guard [логин] [код]` — Передать пятизначный код верификации Steam Guard (с телефона или почты) в активную сессию авторизации бота.
*   `unlock [логин] [AppID]` — Мгновенно разблокировать все достижения в указанной игре (например, `unlock MyBot 730` для CS2).
*   `dlc [логин] [AppID1,AppID2,...]` — Записать в базу данных и применить лицензионные ключи дополнений для выбранного бота фермы.
*   `acf [AppID]` — Сгенерировать чистый, валидный файл манифеста приложения Steam (`appmanifest_ID.acf`) через Walftech Engine.
*   `lua [логин]` — Экспортировать учетные данные бота из SQLite в готовый Lua-скрипт конфигурации для сторонних менеджеров.
*   `collect` — Запустить Advanced Trade Manager для автоматической очистки инвентарей всех ботов и отправки предметов на Главный аккаунт.
*   `помощь` / `help` — Показать полную карту доступных команд ядра.
